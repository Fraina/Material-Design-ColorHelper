Material Design color helper for scss

## install ##

install via npm:
```
$ npm install scss-mdcolor-helper
```

## Usage ##

```
// MDcolor($palette, $tone)
.className {
  color: MDcolor(red)
  background-color: MDcolor(red, A200)
}
```
