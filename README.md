Material-Design-ColorHelper
===========================

Material Design color helper for scss

## Usage ##

```
	color: MDcolor(red)
	background-color: MDcolor(red, A200)
```

## Palettes ##

*   pink
*   purple
*   deep-purple
*   indigo
*   blue
*   light-blue
*   cyan
*   teal
*   green
*   light-green
*   lime
*   yellow
*   amber
*   orange
*   deep-orange
*   brown
*   grey
*   blue-grey
